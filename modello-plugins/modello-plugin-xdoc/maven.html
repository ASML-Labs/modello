Attribute Type Since Description
extender String 3.0.0 The location of the parent project, if one exists. Values from the parent project will be the default for this project if they are left unspecified. The path may be absolute, or relative to the current project.xml file.
Element Type Since Description
parent Parent 4.0.0 Specified which project to extend.
modelVersion String 4.0.0 The version of this model you are using.
groupId String 3.0.0 The primary grouping for your project.
artifactId String 3.0.0 The identifier used when generating the artifact for your project.
type String 4.0.0 The type of artifact this project produces.
Default value is: jar.
name String 3.0.0 Human readable name of the project.
version String 4.0.0 The current version of the project.
shortDescription String 3.0.0 An abbreviated description of the project.
description String 3.0.0 A detailed description of the project. This element is usually specified as CDATA to enable the use of HTML tags within the description. This description is used to generate the front page of the project's web site.
url String 3.0.0 The URL where the project can be found.
logo String 3.0.0 The logo for the project.
issueManagement IssueManagement 4.0.0 The project's issue management information.
ciManagement CiManagement 4.0.0 The project's continuous integration management information.
inceptionYear String 3.0.0 The year the project started.
repositories/repository* List<Repository> 4.0.0 (Many) The lists of the remote repositories
pluginRepositories/pluginRepository* List<Repository> 4.0.0 (Many) The lists of the remote repositories for discovering plugins
mailingLists/mailingList* List<MailingList> 3.0.0 (Many) The mailing lists for the project.
developers/developer* List<Developer> 3.0.0 (Many) This element describes all of the developers associated with a project. Each developer is described by a developer element, which is then described by additional elements (described below). The auto-generated site documentation references this information.
contributors/contributor* List<Contributor> 3.0.0 (Many) This element describes all of the contributors associated with a project who are not developers. Each contributor is described by a contributor element, which is then describe by additional elements (described below). The auto-generated site documentation references this information.
dependencies/dependency* List<Dependency> 3.0.0 (Many) This element describes all of the dependencies associated with a project. Each dependency is described by a dependency element, which is then described by additional elements (described below).
overrides/override* List<Override> 4.0.0 (Many) This element describes all of the dependency overrides for a project. Each dependency is described by a override element, which is then described by additional elements (described below).
licenses/license* List<License> 3.0.0 (Many) This element describes all of the licenses for this project. Each license is described by a license element, which is then describe by additional elements (described below). The auto-generated site documentation references this information. Projects should only list the license(s) that applies to the project and not the licenses that apply to dependencies.
packageGroups/packageGroup* List<PackageGroup> 3.0.0 (Many) Package groups required for complete javadocs.
reports/report* List<String> 3.0.0 (Many) This element includes the specification of reports to be included in a Maven-generated site. These reports will be run when a user executes maven site. All of the reports will be included in the navigation bar for browsing in the order they are specified.
scm Scm 4.0.0 Specification for the SCM use by the project.
builder Build 3.0.0 Information required to build the project.
organization Organization 3.0.0 This element describes various attributes of the organziation to which the project belongs. These attributes are utilized when documentation is created (for copyright notices and links).
distributionManagement DistributionManagement 4.0.0 Distribution information for a project.
local Local 4.0.0 Local configuration information.
properties/key=value* Properties 3.0.0 (Many) Properties about the project. This allows you to configure your project and the plugins it uses.
preGoals/preGoal* List<PreGoal> 4.0.0 (Many) Set of decorator(s) injected before the target goal(s).
postGoals/postGoal* List<PostGoal> 4.0.0 (Many) Set of decorator(s) injected after the target goal(s).

This element describes all of the mailing lists associated with a project. Each mailing list is described by a mailingList element, which is then described by additional elements (described below). The auto-generated site documentation references this information.

Element Type Since Description
name String 3.0.0 The name of the mailing list.

unsubscribe String 3.0.0 The email address or link that can be used to unsubscribe to the mailing list. If this is an email address, a mailto: link will automatically be created when the documentation is created.
post String 4.0.0 The email address or link that can be used to post to the mailing list. If this is an email address, a mailto: link will automatically be created when the documentation is created.
archive String 3.0.0 The link to a URL where you can browse the archive.
